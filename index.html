Padding

El Padding pone un espaciado dentro de los margenes de un borde

Border

Agrega un borde con las caraccteristicas que queramos

Position

Position se encarga de posicionar elementos

Float

La propiedad `float` en CSS permite posicionar un elemento a la izquierda o derecha de su contenedor, haciendo que otros elementos fluyan a su alrededor. Se utiliza comúnmente para crear diseños de columnas y para alinear texto con imágenes. Sin embargo, puede causar problemas de desbordamiento en el contenedor, y hoy en día, hay métodos más modernos, como Flexbox y CSS Grid, que se prefieren para el diseño web.
